fMRIPrep

fMRIPrep is a tool for pre-processing function MRI images, developed in Russ Poldrack's lab.

  • Code at github
  • Docs
  • Publication

fMRIPrep 1.3.1 is implemented on jjm3 as a Singularity container, derived from the Poldrack lab official docker release on dockerhub. The Singularity recipe is on jjm3 at /var/lib/singularity/fmriprep/. The executable (/usr/local/fmriprep) is a wrapper script that maps portions of the local filesystem into the container, and then runs fmriprep on BIDS-compatible files found there.

To run it (sans SLURM for now):

fmriprep /my/bids/input_directory /my/output_directory participant --additional_options

This is the only installation of fMRIPrep, for now.
