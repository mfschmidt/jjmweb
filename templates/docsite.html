JJM Documentation

This django-based documentation site is hosted by apache2, with MySQL, on jjm12.

The code is hosted on github at https://github.com/mfschmidt/jjmweb. If jjm12 is destroyed for any reason, we can easily re-deploy this information elsewhere or on its replacement. This is all publicly available, as nothing on this site is confidential. If that changes, we'll need to remove it from github.

This site was built hastily, meaning that it follows django conventions of having a base template for consistency (/var/www/django/jjmweb/templates/base_template.html), but fails to meet the expectation of storing these documents in the MySQL database. Instead, they are each saved as html files (/var/www/django/jjmweb/templates/). Also, while it should be easily containerized, and has a script to do so, it is currently running directly via jjm12's apache2 installation. See /etc/apache2/sites-available/000-default.conf for configuration.

The configuration points to django code at /var/www/django/jjmweb/, all owned by the webrunner user. MySQL credentials are stored in webrunner's ~/.ssh/my.cnf file.

For help or more details, email Mike Schmidt and cc Frank Gonzales.
