FreeSurfer

FreeSurfer is a tool for creating cortical surface meshes from MRI, along with subcortical segmentations and cortical parcellations. It was developed at The Martinos Center.

  • Code at github
  • Docs
  • Publications

FreeSurfer 6.0.0 is implemented on jjm15 as a Singularity container, derived from the official release on dockerhub. The Singularity recipe is on jjm15 at /var/lib/singularity/freesurfer/. The executable (/usr/local/freesurfer) is a wrapper script that maps portions of the local filesystem into the container, and then runs freesurfer on the supplied T1 image file.

To run it (sans SLURM for now):

freesurfer /my/T1-weighted-image.nii /my/subjects_dir subject_id --additional_options

Non-containerized

FreeSurfer 6.0.0 has not been installed locally on any of these nodes.
